# Создаем карточки с характерисками человека для игры

Этот проект представляет собой простой скрипт на Python для создания игровых карточек в формате .svg. Скрипт создаёт 10 карточек, не изменяя исходный файл шаблона. Значения характеристик создаются рандомайзером на основе библиотеки Faker.

## Установка

1. Убедитесь, что у вас установлен Python 3.
2. Установите необходимые зависимости с помощью pip:
```bash
pip install faker
```
## Использование
1. Скачайте следующие файлы:
    game_cards_players.py - главный файл
    file_operations.py - скрипт, содержащий функцию (charsheet.svg)
    charsheet.svg - шаблон карточки
    skills.txt - список заклинаний и навыков
    letters_mapping.txt - словарь, необходимый для замены обычных символов русского алфавита на рунические
2. Запустите скрипт:
```bash
python ваш_скрипт.py
```
3. Скрипт создаст в директории папку Card_players, в которой создаются новые карточки.
